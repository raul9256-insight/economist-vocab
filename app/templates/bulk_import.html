0. Generate With AI

Use the OpenAI API to generate English definitions, example sentences, and synonyms for missing words automatically.

1. Export Template

Generate an Excel file with word, frequency band, word type, and Chinese definitions already filled in. You can then complete the IPA, English definition, and example fields.

2. Fill In The File

Recommended columns to edit:

  • pronunciation: IPA, for example /əˈbɪləti/
  • english_definition: short English meaning
  • example_sentence: one natural sentence using the word
  • synonyms: optional, one per line or separated by semicolons
  • sentence_distractors: optional wrong example options
  • notes: optional personal memory notes

3. Upload The File

Supported formats: .xlsx and .csv. Excel files can use the exported template columns or the original band workbook columns: Vocabulary, Type of word, English Definition, Chinese Definition, Simplified Chinese Definition, Example Sentence, and IPA.

4. Export Taxonomy Template

Export a progression-mapping template so you can organize basic, exam, business, and AI vocabulary paths in Excel first.

5. Fill In The Taxonomy File

Recommended columns to curate first:

  • cluster_slug and cluster_label: define the meaning family, for example importance
  • stage_rank and stage_label: describe growth such as basic, exam, business, or expert
  • formality_level, precision_level, exam_relevance, business_relevance, ai_relevance: map how the word evolves across contexts
  • related_word and relation_type: link upgrade words such as important -> crucial
  • relation_explanation and usage_note: explain why the target word is not just harder, but different in use

6. Upload The Taxonomy File

Supported formats: .xlsx and .csv.
